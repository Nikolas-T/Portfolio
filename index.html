Books vs Movies

BooksVsMovies is powered by the OpenLibrary Api and TheMovieDB. The OpenLibrary Api was used to get the title info, the author, the published date, and the cover for all the books searched. TheMovieDB was used to get the title info, the release date, and the poster for all the movies searched. The styling was done with Css, Bulma, Font-Awesome, as well as Adobe Illustrator. BooksVsMovies uses the jquery library for the javascript.
